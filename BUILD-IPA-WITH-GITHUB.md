# بناء IPA من GitHub ثم توقيعه في GBox

هذه النسخة مجهزة لتبني ملف IPA غير موقّع تلقائيًا باستخدام GitHub Actions على جهاز Mac سحابي.

## طريقة الاستخدام

1. أنشئ Repository جديد في GitHub.
2. ارفع **كل محتويات هذا المجلد** كما هي، بما فيها المجلد المخفي `.github`.
3. افتح تبويب **Actions** في GitHub.
4. اختر **Build FitbitAir IPA**.
5. اضغط **Run workflow** ثم أكد التشغيل.
6. انتظر حتى تصبح العملية خضراء ✅.
7. افتح نتيجة التشغيل، وانزل إلى قسم **Artifacts**.
8. حمّل الملف **FitbitAir-Unsigned-IPA**.
9. فك ملف الـArtifact المضغوط؛ ستجد داخله:
   `FitbitAir-Unsigned.ipa`
10. افتح ملف IPA في GBox ووقّعه بشهادتك ثم ثبته.

## مهم

- ملف IPA الناتج **غير موقّع** عمدًا، حتى تقدر توقّعه داخل GBox.
- لا تحتاج Mac أو Xcode للتشغيل المعتاد.
- عند تعديل ملفات التطبيق داخل `FitbitAir-iOS`، سيبدأ بناء جديد تلقائيًا.
- تقدر أيضًا تشغيل البناء يدويًا في أي وقت من زر **Run workflow**.
- ملفات Railway موجودة كما هي ولم يتم حذفها.

## إذا ظهر خطأ

أرسل لقطة شاشة من صفحة GitHub Actions التي يظهر فيها السطر الأحمر، وبنصلح نفس ملف المشروع.
