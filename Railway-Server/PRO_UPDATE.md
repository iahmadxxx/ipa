# Professional Workout Update

## New features
- Add exercises from Telegram inside each workout day.
- Delete exercises from Telegram with confirmation.
- Historical workout records remain safe when an exercise is deleted from the plan.
- Exercise plan changes are stored automatically in the existing SQLite database.
- Edit any set from the Mini App (weight and reps).
- Redesigned workout Mini App with a cleaner professional interface.
- Redesigned Telegram workout menus and button organization.

## Deployment
Upload all files to the same GitHub repository and let Railway redeploy.
No new environment variables, database service, domain, or manual migration are required.
