# تحديث التجديد التلقائي للتوكن (reauth) 🔑

## وش الجديد

1. **أمر `/reauth` (أو `/تجديد`)** — يرسلك زر، تضغطه → صفحة موافقة جوجل → تضغط "السماح" → **البوت يستلم التوكن الجديد ويحفظه بنفسه**. بدون Playground، بدون نسخ لصق، بدون فتح Railway. ⏱️ 15 ثانية.

2. **تنبيه تلقائي** — إذا انتهى التوكن (كل 7 أيام)، أي أمر تجربه يرسلك زر التجديد تلقائيًا بدل رسالة خطأ غامضة. حتى الرسالة الصباحية لو فشلت بسبب التوكن، توصلك رسالة "جدد من هنا".

3. **التوكن صار يخزن بقاعدة البيانات** (نفس الـ Volume حق التمارين) — يعني يصمد مع أي إعادة نشر. متغير `GOOGLE_REFRESH_TOKEN` صار مجرد قيمة أولية احتياطية.

4. **إصلاح جانبي مهم:** خدمات الـ Cron (الملخص الصباحي والتنبيهات) صارت تشتغل **داخل خدمة الويب** عبر رابط، بدل خدمات معزولة — وهذا يحل مشكلة كانت موجودة عندك أصلاً: خدمات الـ Cron ما كانت تشوف بيانات تمارينك (الـ Volume مربوط بخدمة الويب بس!) ولا بتشوف التوكن المتجدد.

## خطوات التركيب

### 1. ارفع الملفات المحدثة لـ GitHub
الملفات المتغيرة/الجديدة:
- `token_store.py` (جديد)
- `google_health_client.py` (محدث)
- `webhook_app.py` (محدث)

بدّلها بمستودعك → Commit → Railway ينشر تلقائيًا.

### 2. ضيف رابط إعادة التوجيه بـ Google Cloud (مرة وحدة)
1. روح [console.cloud.google.com](https://console.cloud.google.com) → مشروعك → **APIs & Services** → **Credentials**.
2. اضغط على الـ OAuth Client حقك.
3. تحت **Authorized redirect URIs** اضغط **+ ADD URI** وضيف:
   ```
   https://<رابط-Railway-حقك>/reauth/callback
   ```
   (مثال: `https://web-production-45a08f.up.railway.app/reauth/callback`)
4. **Save**.

بدون هالخطوة جوجل بترفض إعادة التوجيه بخطأ redirect_uri_mismatch.

### 3. حدّث خدمات الـ Cron (تصير أبسط!)
بعد النشر، ابعث بالمتصفح: `https://<رابط-Railway>/cron/key` — بيرسل لتلقرامك مفتاح سري.

بعدها بكل خدمة Cron بـ Railway → Settings → **Start Command** بدّله:

- خدمة الملخص الصباحي:
  ```
  curl -s "https://<رابط-Railway>/cron/daily?key=<المفتاح>"
  ```
- خدمة التنبيهات:
  ```
  curl -s "https://<رابط-Railway>/cron/alerts?key=<المفتاح>"
  ```

(نفس الجداول الزمنية الحالية، بس الأمر تغير). وميزتها: ما تحتاج تضيف أي متغيرات للخدمتين بعد اليوم — كل شي يشتغل بخدمة الويب.

### 4. جرب!
- ابعث `/reauth` للبوت → اضغط الزر → وافق → لازم توصلك رسالة "✅ تم تجديد ربط Google Health بنجاح".
- جرب `/today` بعدها للتأكد.

## ملاحظة أمنية
رابط التجديد محمي بتوقيع مشتق من توكن بوتك — ما ينفع أحد غريب يستخدمه لاستبدال توكنك. ومع كذا لا تشارك روابط `/cron/...?key=` مع أحد.
